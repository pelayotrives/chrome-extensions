# Chrome Extensions

A collection of Google Chrome extensions built on **Manifest V3**.

## Included Extensions

### [Crop & Convert](/crop-and-convert)

Lightweight image processing tool. Crop images to 1:1 or 16:9 and convert JPG/PNG to **WEBP**, individually or in batch.

### [Extensio Patronus](/extensio-patronus)

Extension management hub. Create groups, list, search, enable/disable, and open options for other extensions from a single popup.

## Local Installation

1. Open `chrome://extensions/` in your browser
2. Enable **Developer mode** (top right corner)
3. Click **Load unpacked**
4. Select the folder of the extension you want to try
